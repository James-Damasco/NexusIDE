# Nexus IDE 🌌

Nexus IDE is a state-of-the-art, fully responsive, browser-based code editor. Designed with a futuristic dark theme, neon accents, and a glassmorphism UI, it brings a professional desktop IDE experience directly to your web browser without requiring any backend or server setup.
---
## ✨ Features

- **Advanced Code Editor:** Powered by Monaco Editor (the engine behind VS Code), featuring syntax highlighting, auto-completion, multiple cursors, and minimap support.

- **Live Web Preview:** Instantly preview your HTML/CSS/JS combinations inside an integrated, sandboxed iframe.

- **Integrated Terminal:** A simulated command-line interface powered by xterm.js for a true developer environment feel.

- **AI Assistant Panel:** A sleek UI for an AI coding assistant, ready for your custom LLM backend integrations.

- **Fully Responsive:** Adapts seamlessly across desktop, tablet, and mobile devices with tailored navigation paradigms.

- **Dynamic Layout:** Resizable and collapsible workspace panels using Split.js.

- **Command Palette:** Quick-access command launcher triggered via Ctrl+P or Cmd+P.
---
# 🛠️ Technologies Used

- Core: **HTML5, CSS3, Vanilla JavaScript (ES6+)**

- Styling: **Tailwind CSS**

- Editor Engine: **Monaco Editor**

- Terminal Engine: **xterm.js**

- Layout Management: **Split.js**

- Icons: **Lucide Icons**

- Animations: **GSAP**

# 🚀 Getting Started

Since Nexus IDE is a 100% client-side application, getting started is incredibly simple:

- Download [NexusIDE](https://github.com/James-Damasco/NexusIDE.git) file.

- Open the index.html file directly in any modern web browser (Chrome, Firefox, Safari, Edge).

**Start coding!**

**(Note: To avoid CORS issues with Monaco Editor's web workers when running locally from a file:// protocol, the project utilizes inline data URIs for its web workers, ensuring a plug-and-play experience).**
---

# 📱 Mobile Experience

Nexus IDE includes a specifically tailored mobile experience. When viewed on smaller screens, the complex multi-panel desktop UI automatically transitions into a mobile-friendly application layout. A bottom navigation bar replaces the resizable sidebars, allowing you to seamlessly toggle between the Editor, File Explorer, Terminal, and AI Assistant without cluttering your screen real estate.
---
# 🤝 Customization

The IDE is built to be easily customizable. You can adjust the color scheme by modifying the Tailwind config in the <head> of the HTML file, specifically tweaking the workspace, panel, accent, and primary color variables.
